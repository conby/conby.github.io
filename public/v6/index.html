.js

c3.js is a pure javascript and pure asynchronous data application which aims to train and adapt content according to C3 big data platform. It works with:

  • jquery.js
  • d3.js
  • joint.js
  • bootstrap.js
  • ...

This application is entirely free to use, and c3.js is incompatible with all old C3 versions, including v3.x LTS and v5.x LTS, it works only for C3 v6.x(named B2 currently), but you're always welcome to try out offical website conby.com for more information.

Currently tested:

  • PC(IE9+)
  • IOS/iPad
  • Android/Pad
  • Cordova
  • Unity3D View
  • Wechat
